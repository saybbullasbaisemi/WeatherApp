## ✨ Features
* **Real-time Updates:** Current weather details-ai API vazhiya fetch pannum.

* **Easy Search:** City name pottu search panna weather details kedaikkum.

* **Frontend:** HTML, CSS
* **API:** OpenWeatherMap API
* **Hosting:** PythonAnywhere
